> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps `- [ ]`.

**Goal:** Ajouter les vaisseaux au simulateur : une **3e file parallèle** (on bâtit + recherche + produit des vaisseaux en même temps), puits de ressources/temps, avec le jalon « 1er vaisseau » réel (un vaisseau produit).

**Architecture:** Étend `packages/game-sim`. Les vaisseaux ne réinjectent PAS dans l'économie (puits pur) → pas de boucle de bonus comme la recherche. File parallèle au build et à la research, gated par le niveau de `shipyard`.

## Référence (game-engine)
- `unitCost(def: { cost: {minerai;silicium;hydrogene} }): ResourceCost` (coût FIXE par unité, pas de scaling).
- `unitTime(def, bonusMultiplier, timeDivisor=2500): number` = `floor(((minerai+silicium)/2500)*3600*bonusMultiplier)`, min 1s.
- `SHIPS` (dans `seed-game-config.ts`, à extraire) : chaque vaisseau a `costMinerai/Silicium/Hydrogene`, `prerequisites: { buildings:[{buildingId;level}], research:[{researchId;level}] }`, `categoryId`, `role`. 1er buildable = **`prospector`** (prereq `shipyard` L2, aucune recherche).

## État de départ (sur main, recherche déjà livrée)
`SimState` a `build`/`research` (files parallèles). `SimEngine` a `startBuild`/`startResearch`, `nextEventIn` (min des délais positifs des 2 files), `advance` (finalise chaque file due). `config.ts` : loadBuildings/loadProductionConfig/loadResearch/loadBonuses (data dans `packages/db/src/game-config-data.ts`, NB : `SHIPS` PAS encore extrait).
